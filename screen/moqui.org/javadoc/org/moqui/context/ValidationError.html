[Java] Class ValidationError

              • org.moqui.context.ValidationError 

              • @java.lang.SuppressWarnings("unused")
public class ValidationError
extends BaseArtifactException

                ValidationError - used to track information about validation errors. This extends the BaseException and has additional information about the field that had the error, etc. This is not generally thrown all the way up to the user and is instead added to a list of validation errors as things are running, and then all of them can be shown in context of the fields with the errors.
